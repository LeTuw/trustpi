Get Wallet Trust Score

GET
/api/pi-trust/<wallet_id>

Retrieve a comprehensive trust score for any Pi Network wallet address. The score is calculated based on multiple factors including transaction history, account age, balance stability, and network connections.

Authentication

This endpoint requires API key authentication. Include your API key in the request headers.

# Required headers
Authorization: Bearer your_api_key_here
# OR
X-API-Key: your_api_key_here

Parameters

Parameter Type Required Description
wallet_id String Required The Pi Network wallet address to analyze (e.g., GBQWCPVKFHB35TANHS7EBVELFGXCDGG6UPVYAZGH2MDT42PFG23EQUSN)

Request Example

# cURL example
curl -X GET \
  "https://trustpi.fit/api/pi-trust/GBQWCPVKFHB35TANHS7EBVELFGXCDGG6UPVYAZGH2MDT42PFG23EQUSN" \
  -H "Authorization: Bearer pk_live_your_api_key_here" \
  -H "Content-Type: application/json"
# JavaScript example
const response = await fetch(
  'https://trustpi.fit/api/pi-trust/GBQWCPVKFHB35TANHS7EBVELFGXCDGG6UPVYAZGH2MDT42PFG23EQUSN',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer pk_live_your_api_key_here',
      'Content-Type': 'application/json'
    }
  }
);

const data = await response.json();

Response Fields

wallet_id String
The Pi wallet address that was analyzed
pi_trust_score Integer
Overall trust score (300-1000). KYC-verified wallets start at 300.
category String
Trust category based on score: "Unproven", "Newcomer", "Active Participant", "Established Member", "Reliable Partner", "Trusted Leader", "Elite"
components Object
Detailed breakdown of individual component scores (0-1000)
breakdown Object
Detailed analysis including weighted scores and interpretation

Error Responses

Status Code Description
400 Bad Request - Invalid wallet address format
401 Unauthorized - Missing or invalid API key
403 Forbidden - Plan is not active
404 Not Found - Wallet address not found on Pi Network
429 Too Many Requests - Monthly request limit exceeded
500 Internal Server Error - Trust score calculation failed

Rate Limiting

Rate limits are based on your subscription plan:

  • Starter: 500 requests/month
  • Growth: 5,000 requests/month
  • Professional: 20,000 requests/month
  • Enterprise: Unlimited requests
